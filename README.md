# Market Management System

Console-based market management application written in C#.

This project allows managing products and sales with full
business validation and stock consistency.

---

## Features

### Product Management
- Add, update and delete products
- List all products
- Filter products by:
  - Name
  - Category
  - Price range

### Sale Management
- Create sales with multiple products
- Validate stock availability
- Calculate total sale amount
- Delete sales and restore stock
- Return products from a sale
- Filter sales by:
  - Amount range
  - Date
  - Date range

---

## Technologies
- C#
- .NET Console Application
- LINQ
- OOP (Interfaces, Inheritance, Encapsulation)

---

## Notes
- Product stock is automatically updated after sales
- Invalid inputs are handled with validations
- Business logic is separated from menu logic
